# Arbitration Art

Arbitration Art is a full-stack platform managing automated cryptocurrency arbitrage strategies.

## Tech Stack
- **Frontend:** Vue 3, Quasar Framework, TypeScript, SASS.
- **Backend:** Python 3.12, Django 6.0, Django REST Framework.
- **State Management:** Pinia / VueX (Quasar Stores).
- **Styling:** Quasar UI components + Custom SASS variables.
